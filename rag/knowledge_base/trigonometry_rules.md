# Trigonometry Rules for JEE Mathematics

## 1. Basic Trigonometric Ratios

For angle θ in a right triangle:

sin θ = opposite / hypotenuse
cos θ = adjacent / hypotenuse
tan θ = opposite / adjacent

Reciprocal identities:

cosec θ = 1 / sin θ
sec θ = 1 / cos θ
cot θ = 1 / tan θ

---

## 2. Fundamental Identity

sin²θ + cos²θ = 1

From this:

1 + tan²θ = sec²θ

1 + cot²θ = cosec²θ

---

## 3. Complementary Angle Identities

sin(90° − θ) = cos θ

cos(90° − θ) = sin θ

tan(90° − θ) = cot θ

sec(90° − θ) = cosec θ

cosec(90° − θ) = sec θ

cot(90° − θ) = tan θ

---

## 4. Trigonometric Values of Standard Angles

θ      sinθ    cosθ    tanθ

0°     0       1       0

30°    1/2     √3/2    1/√3

45°    1/√2    1/√2    1

60°    √3/2    1/2     √3

90°    1       0       undefined

---

## 5. Angle Addition Formulas

sin(A + B) = sinA cosB + cosA sinB

sin(A − B) = sinA cosB − cosA sinB

cos(A + B) = cosA cosB − sinA sinB

cos(A − B) = cosA cosB + sinA sinB

tan(A + B) = (tanA + tanB) / (1 − tanA tanB)

tan(A − B) = (tanA − tanB) / (1 + tanA tanB)

---

## 6. Double Angle Formulas

sin(2θ) = 2 sinθ cosθ

cos(2θ) = cos²θ − sin²θ

Also:

cos(2θ) = 2cos²θ − 1

cos(2θ) = 1 − 2sin²θ

tan(2θ) = 2tanθ / (1 − tan²θ)

---

## 7. Triple Angle Formulas

sin(3θ) = 3sinθ − 4sin³θ

cos(3θ) = 4cos³θ − 3cosθ

tan(3θ) = (3tanθ − tan³θ) / (1 − 3tan²θ)

---

## 8. Half Angle Formulas

sin²(θ/2) = (1 − cosθ) / 2

cos²(θ/2) = (1 + cosθ) / 2

tan(θ/2) = sinθ / (1 + cosθ)

Also:

tan(θ/2) = (1 − cosθ) / sinθ

---

## 9. Product to Sum Identities

sinA sinB = 1/2 [cos(A − B) − cos(A + B)]

cosA cosB = 1/2 [cos(A − B) + cos(A + B)]

sinA cosB = 1/2 [sin(A + B) + sin(A − B)]

---

## 10. Sum to Product Identities

sinA + sinB = 2 sin((A+B)/2) cos((A−B)/2)

sinA − sinB = 2 cos((A+B)/2) sin((A−B)/2)

cosA + cosB = 2 cos((A+B)/2) cos((A−B)/2)

cosA − cosB = −2 sin((A+B)/2) sin((A−B)/2)

---

## 11. Important Transformations

1 − cosθ = 2 sin²(θ/2)

1 + cosθ = 2 cos²(θ/2)

sin²θ = (1 − cos2θ)/2

cos²θ = (1 + cos2θ)/2

These are commonly used in JEE simplification problems.

---

## 12. Trigonometric Equations

General solutions:

sin θ = 0
θ = nπ

cos θ = 0
θ = (2n + 1)π/2

tan θ = 0
θ = nπ

where n is an integer.

---

## 13. Inverse Trigonometric Functions

sin⁻¹x range:
−π/2 ≤ sin⁻¹x ≤ π/2

cos⁻¹x range:
0 ≤ cos⁻¹x ≤ π

tan⁻¹x range:
−π/2 < tan⁻¹x < π/2

---

## 14. Important Inverse Trig Identities

sin⁻¹x + cos⁻¹x = π/2

tan⁻¹x + cot⁻¹x = π/2

sin⁻¹x = tan⁻¹(x / √(1 − x²))

cos⁻¹x = tan⁻¹(√(1 − x²) / x)

---

## 15. Trigonometric Substitution (Useful in Calculus)

√(a² − x²) → x = a sinθ

√(a² + x²) → x = a tanθ

√(x² − a²) → x = a secθ

These substitutions simplify many calculus problems.
